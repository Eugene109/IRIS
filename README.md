# IRIS
Independent Robotic Import System

A PCB drone swarm architecture designed to autonomously deliver Skittles, named after the goddess of the rainbow.

## Highlights
 - STM32H743 MCU clocked at 480 MHz
 - 2MB Flash + 1MB RAM
 - High-voltage MOSFET ESCs with sensorless drive
 - XT60 battery connector
 - 2S to 3S battery voltages
